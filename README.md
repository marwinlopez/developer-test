# developer-test.

## Descarga



Si tienes git instalado en tu PC, sólo tienes que hacer lo siguiente:

```
$ git clone https://github.com/marwinlopez/developer-test.git
```

## Configuración App


```
$ cd developer-test
$ cd server
$ npm install
```

```
$ cd developer-test
$ cd client
$ npm install
```

## Docker-compose
```
$ cd developer-test
$ docker-compose build
$ docker-compose up
```
